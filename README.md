# 今天吃什么 🍜

一个有趣的美食随机选择器，帮你解决"今天吃什么"的选择困难症！

## ✨ 功能特点

- 🎲 **随机选择**：从美食库中随机推荐美食
- ⏰ **时间段分类**：按早/中/下午茶/晚/宵夜严格分类，默认选中当前时段
- 🔍 **搜索功能**：支持按名称、分类、描述搜索美食
- 🌙 **暗色模式**：支持明暗主题切换，保护眼睛
- 🎊 **动画效果**：流畅的转场动画和选中庆祝效果
- 📱 **响应式设计**：完美适配桌面和移动设备
- 😠 **愤怒模式**：连续选择困难5次后的特殊彩蛋

## 🛠️ 技术栈

- **前端框架**：React 19 + TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **动画库**：Framer Motion
- **图标库**：Lucide React
- **工具库**：clsx, tailwind-merge
- **彩蛋效果**：canvas-confetti

## 🎨 主题配置

项目支持明暗两种主题模式：
- 亮色模式：橙色系配色，温暖活泼
- 暗色模式：深色系配色，护眼舒适

主题切换通过在 `document.documentElement` 上添加/移除 `dark` 类来实现。

## 🌐 部署配置

项目配置了 `base: '/eat/'` 路径前缀，方便部署到 nginx 子目录。

### Nginx 配置示例

```nginx
location /eat/ {
    alias /path/to/dist/;
    try_files $uri $uri/ /eat/index.html;
}
```

## 🎮 使用说明

1. **选择时段**：顶部下拉选择器可切换不同时段（早/中/下午茶/晚/宵夜/全部）
2. **查看当前时段**：右上角显示当前时间段的emoji和名称
3. **开始随机**：点击"开始随机"按钮，系统会从当前选中时段的美食中随机推荐
4. **查看结果**：滚动停止后显示最终选择的美食
5. **搜索美食**：在底部搜索框输入关键词筛选美食
6. **切换主题**：点击右上角的主题切换按钮
7. **重试次数**：右上角显示当前重试次数，最多5次

## 🕐 时间段分类

- **早上** (6:00-11:00)：豆浆油条、包子、煎饼果子、广式早茶、小笼包等
- **中午** (11:00-14:00)：兰州拉面、红烧肉、水煮鱼、酸菜鱼、麻婆豆腐等
- **下午茶** (14:00-17:00)：奶茶、蛋糕甜点、冰淇淋、水果捞、咖啡等轻食甜品
- **晚上** (17:00-21:00)：火锅、北京烤鸭、新疆大盘鸡、潮汕牛肉火锅、韩式烤肉等
- **宵夜** (21:00-6:00)：小龙虾、羊肉串、螺蛳粉、炸鸡、韩式炸鸡等
- **全部**：包含所有时段的美食
